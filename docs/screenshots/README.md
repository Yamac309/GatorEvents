# Screenshots

Drop your screenshots in this folder and reference them from the root `README.md`.

Recommended captures (1280×800 or higher, PNG):

| File | What to capture |
|---|---|
| `feed.png` | The `/` feed view with several events listed, a filter chip selected |
| `map.png` | The `/map` view with the satellite tiles and multiple colored pins |
| `submit.png` | The `/submit` form, ideally with the Places autocomplete dropdown showing |
| `admin-list.png` | The `/admin` dashboard showing the pending-review list |
| `admin-edit.png` | An expanded admin edit panel with the drag-pin mini-map visible |
| `mobile.png` | The app installed as a PWA on a phone (Add to Home Screen) |

Tip: Use your OS screenshot tool (Win+Shift+S on Windows) and crop to just the browser content (no taskbar). PNGs render crisp on GitHub.
